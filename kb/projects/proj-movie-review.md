# Movie Review Analyzer (Personal, 2018)

**What:** Automatic classification of movie reviews using classical NLP.

## What was built

- Bag-of-words text representation
- Seven different ML classifiers with ensemble voting for final prediction
- Automatically sorts and classifies movie reviews as positive/negative

Early NLP project exploring classical text classification before transformer models. Demonstrates foundation in text preprocessing, feature engineering (BoW), and ensemble methods.
