# Live Video Streaming Service (Personal, Jun–Jul 2017)

**What:** eLearning platform supporting live video streaming from teacher to students and real-time chat.

## What was built

- Live video streaming from teacher to all connected students
- Real-time chat between all logged-in users
- Built in 2017 as an early full-stack web project

Early project demonstrating interest in real-time systems and educational technology — a theme that continued with the EdTech TULNA multi-agent evaluation system in 2025.
